Employee Management System
Overview
This project is an Employee Management System implemented in Java using Hibernate. The application allows users to perform CRUD (Create, Read, Update, Delete) operations on employee records.
Features
* Add, update, and delete employee records.
* View the list of employees.
Technologies Used
? Java
? Hibernate
? MVC
? POSTGRESQL  (as the database)
